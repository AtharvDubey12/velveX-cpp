Velvet Syntax

Velvet is a custom mathematical syntax made to simplify and speed up writing LaTeX, by not writing LaTeX at all. Designed for compact yet readable expressions of mathematical operations.

Integration Syntax

Use Integ[order][function][variables][limits] to perform definite or indefinite integration.

Format: 
Integ[1][x^2][x][0->a]
Meaning: Integrate xÂ² once with respect to x from 0 to a.
Rendered as: 
 \[ \int_{0}^{a} x^2 \, dx \]

Note: In case limits are not given, the integral is considered as indefinite.

Differentiation Syntax

Use Diff[order][function][variables] for ordinary derivatives.

Fraction Syntax

Use Frac[numerator][denominator] to represent a fraction.

Frac[cot(x)][tan(y)]]
Meaning: Divide cot(x) by tan(x). 
Frac[2][x^2 * y + sin(x)][x]

Square Root Syntax

Use Sqrt[degree][function] to represent the function with a square root symbol.

Sqrt[a][b]]
Meaning: Represents b^(1/a). 
Sqrt[b]
Meaning: Represents b^(1/2).

Matrix Syntax

Use Mx[[Row 1 in CSV format],[Row 2 in CSV format],...] to represent a M x N order matrix.

Mx[[1,0,0],[0,1,0],[0,0,1]]
Meaning: Represents Identity Matrix. 
Mx[[0,0,0]]
Meaning: Represents [0,0,0] Null vector.
